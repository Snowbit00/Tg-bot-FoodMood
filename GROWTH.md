# Чек-лист продвижения канала

Бот уже работает на рост: кнопки «Подписаться»/«Поделиться» под постами, вотермарка
@канала на фото, призыв в тексте, еженедельный опрос. Ниже — то, что делаете вы.

## 0. Упаковка канала (один раз, бесплатно — сделать в первую очередь)
- [ ] Понятный `@username` и название с ключевым словом («Рецепты», «Готовим дома»).
- [ ] Описание с ключевыми словами (для поиска внутри Telegram) + что получает подписчик + частота постов.
- [ ] Аппетитный аватар (фото блюда или простой логотип).
- [ ] Закреплённый приветственный пост: о чём канал + 3–5 лучших рецептов ссылками.
- [ ] **Включить реакции** (Управление каналом → Реакции).
- [ ] **Включить комментарии** (привязать чат-обсуждение) — без них ER низкий.

## 1. Бесплатный трафик
- [ ] **Взаимопиар (ВП):** написать 15–30 админам кулинарных каналов близкого размера,
      предложить взаимные репосты. Делать пачками каждую неделю.
- [ ] **Папки-подборки (folders):** объединиться с дружественными каналами в папку.
- [ ] **Каталоги:** добавить канал в каталог TGStat, tlgrm.ru и подобные.
- [ ] **Кросс-постинг рецептов:** Pinterest (топ для рецептов!), VK, Дзен, а также короткие
      видео готовки в Reels / Shorts / TikTok. В профиле/описании — ссылка на канал.
      Один рецепт = пост в TG + пин в Pinterest + Reels. Контент уже есть, переупаковывайте.
- [ ] **Тематические чаты:** помогать/отвечать по делу (без спама), ссылка в профиле.

## 2. Платный трафик
- [ ] **Посевы через Telega.in:** каналы по кулинарии с ER ≥ 8–10%, бюджет от ~1–3 тыс ₽
      на тест. Креатив = лучший рецепт + крючок («сохрани, чтобы не потерять»).
      Маркировка рекламы обязательна по закону — биржа делает её сама.
- [ ] **Прямые закупки** у админов — дешевле бирж, но проверяйте накрутку в TGStat
      (резкие скачки подписчиков, низкий ER, боты в комментариях).
- [ ] **Telegram Ads** (официальный кабинет) — для масштаба, выше порог входа.

## 3. Метрики и итерация
- [ ] Раз в неделю смотреть в TGStat / Телеметр: прирост, ER, охваты, **стоимость
      подписчика (CPF)**.
- [ ] Лить бюджет туда, где подписчик дешевле и живее. ER 3–6% — норма, выше — отлично.
- [ ] Когда наберётся ~1000+ живых подписчиков с нормальным ER — регистрировать канал
      на Telega.in уже как площадку и продавать рекламу (ориентир CPM кулинарии — 5–6к₽).

## Реалистичные ожидания
Первые заметные результаты — через 1–2 недели системной работы. Рост = контент (есть) +
бесплатные методы (постоянно) + точечные посевы (по бюджету). Без рывков, но предсказуемо.
